Introduction

A computer is an electronic device that processes data to perform various tasks. It can store, retrieve, and process information, making it a versatile tool for both personal and professional use.

Components of a Computer

A computer consists of several key components, including: Central Processing Unit (CPU) Memory (RAM) Storage (Hard Drive, SSD) Input Devices (Keyboard, Mouse) Output Devices (Monitor, Printer) Motherboard Graphics Processing Unit (GPU)
